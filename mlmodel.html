Event Detection Model (Peak Finding)

A dataset from a time series analysis serves as the model's input. Peak finding was used as the algorithm for the model. In essence, it plots the quantity of tweets in proportion to the posting date. Marks indicating the peak of the plots were the essential modification.

The Date column from the dataset served as the model's predictor variable, and the Number of Tweets column served as the model's target variable.

The analysis used the minimal value for the evaluation matrix since there wasn't an excessive amount of data in the dataset. Without taking into account the plateaus, it scans the plot for pronounced peaks.

The model's performance is in line with what was anticipated. It demonstrates the increase in tweets during several months, particularly during the 2022 election campaign.
